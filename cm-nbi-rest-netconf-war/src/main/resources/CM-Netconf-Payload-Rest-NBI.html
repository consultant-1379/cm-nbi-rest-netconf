/enm/nbi/cm/v1/networkelement/{network-element-name} patch

patch

CM Netconf Payload REST Northbound interface provides support for NBI client applications to apply Netconf configurations on the node, using a REST interface. This rest end point can also be used to perform action on the node and retrieve configuration data from node.
